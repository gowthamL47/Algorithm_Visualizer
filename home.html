Sorting Algorithms

Sorting algorithms arrange elements in a specific order (ascending/descending). Common sorting techniques:

  • Bubble Sort: Repeatedly swaps adjacent elements if they are in the wrong order.
  • Selection Sort: Finds the smallest element and swaps it with the first element.
  • Insertion Sort: Builds the sorted list one element at a time.
  • Merge Sort: Uses divide and conquer to sort the array efficiently.
  • Quick Sort: Selects a pivot and partitions the array recursively.
  • Heap Sort: Utilizes a heap data structure to sort elements.

  • Depth-First Search (DFS): Explores as deep as possible before backtracking.

Binary Search Tree (BST)

A hierarchical data structure where each node has at most two children.

  • Insertion: Adds an element while maintaining BST properties.
  • Search: Finds an element using a divide-and-conquer approach.
  • Deletion: Removes an element while preserving BST structure.
  • Traversal: Preorder, Inorder, and Postorder traversal methods.
Explore Search Algorithms

Trie (Prefix Tree)

Efficient for searching words in dictionaries and auto-complete systems.

  • Insertion: Adds a word character by character.
  • Search: Checks if a word exists in the trie.
  • Deletion: Removes a word while maintaining efficiency.
  • Prefix Search: Finds all words starting with a given prefix.
